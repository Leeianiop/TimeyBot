# TimeyBot
## A cute IoT time tracker, functions includes:
-Stopwatch,
-Timer,
-Local Time,
-Calender reminder,
-Alarm,
-Music Player,
-Pomodoro Tracker
## Why Timey
The button on top of the case can be customized to any head; this file includes a dog head and a cat head
It features all-in-one functions and can be built on to add your own Songs, Functions, Design and more
Also doubles as an study partner, with time tracking and task reminder features and no distractions from other apps.
## Required parts: 
4-inch OLED display, ESP32, 4ohm square speaker, Button: check BOM for price list
Stl files include: TimeyBot MAIN 2, TimeyBot Slider, backplate, HEADS file>> dog, cat
